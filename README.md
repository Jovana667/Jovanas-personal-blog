# Jovanas-personal-blog

## Description

This personal blog application allows users to create and view all input blog posts. It includes features like a light/dark mode toggle and local storage for post data.

## Features

- Blog post data is stored in the browser's localStorage as a JSON array
- Each post contains:
  - Author's username
  - Post title
  - Post content
- The application uses JavaScript for all functionality, including form handling and page navigation.

## How to Use

1. **Creating a Blog Post**:
   - On the landing page, fill out required fields on the form with your username, blog title, and content
   - A prompt will appear if any fields within the form are left incomplete
   - Click submit to store the post and to be redirected to the blog page

2. **Viewing Blog Posts**:
   - Navigate to the posts page to see all blog entries containing the title, content and username, in order from first submission to last.

3. **Using Light/Dark Mode**:
   - Click the light/dark mode toggle in the header to switch between themes which will remain when navigating between pages, unless switched by the user.

4. **Navigating the App**:
   - Use the "Back" button on the posts page to return to the landing page to create more entries

5. **Accessing Developer Information**:
   - Click the portfolio link in the footer to be redirected to my portfolio
